
# JAVA_TICKET_OFFICE

## 주제
놀이공원 티켓 발급 프로그램

## 클래스 구조
![슬라이드2](https://user-images.githubusercontent.com/46421950/115478597-7b6aa680-a281-11eb-9904-406e882df947.JPG)

## 기능
1. 입력1 : 주야간권 선택
2. 입력2 : 주민등록번호 입력
3. 입력3 : 티켓 매수 입력
4. 입력4 : 우대사항 선택
5. 출력1 : 티켓 가격 출력
6. 출력2 : 티켓 총 가격 출력

### 파일 변수 위치
- ConstValue의 `fileOutput`변수에 저장된 "report.csv"
